# 経理部 - 財務管理 & レポーティング

## ミッション
全取引の収支を正確に記録し、月間利益目標100万円の達成を数字で支える。

## 担当業務
1. **売上記録** - eBay売上の日次記録
2. **仕入れ記録** - 仕入れコストの正確な記録
3. **利益計算** - 手数料・為替・還付込みの実利益算出
4. **日次レポート** - 毎朝自動レポート出力
5. **消費税還付** - 還付申請用データ整備
6. **確定申告準備** - 年間収支データの整理

## 月間KPI
| 項目 | 目標 |
|------|------|
| 月間利益 | ¥1,000,000（年内達成） |
| ROI | 25%以上 |
| 為替レート | 155 JPY/USD（定期見直し） |

## 計算ルール
```
売上(JPY) = eBay売上(USD) × 為替レート
eBay手数料 = 売上 × 13.25%（FVF）
海外手数料 = 売上 × 1.35%
Payoneer手数料 = 売上 × 2%
Promoted手数料 = 売上 × 3%
手数料合計 = 売上 × 19.6%
消費税還付 = 仕入れ価格 × 10%
純利益 = 売上 - 手数料合計 - 仕入れ価格 + 消費税還付
```

## レポート体系
- **日次**: 毎朝自動出力（売上・利益・出品数サマリー）
- **週次**: 月曜CEO提出（部署別KPI・P/L）
- **月次**: 月末締め（確定収支・ROI・在庫評価）

## 使用ツール
- `daily_report.py` - 日次レポート
- `sheets_manager.py` - スプレッドシート管理
