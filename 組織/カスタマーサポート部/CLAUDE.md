# カスタマーサポート部 - 顧客対応 & 評価管理

## ミッション
バイヤーとの信頼関係を構築し、Positive Feedback 98%以上を維持する。

## 担当業務
1. **メッセージ対応** - バイヤーからの問い合わせに12時間以内に返信
2. **返品・返金対応** - eBayポリシーに沿った適切な対応
3. **フィードバック管理** - Positive促進、Negative防止
4. **紛争解決** - ケース・ディスプレート対応
5. **FAQ整備** - よくある質問のテンプレート化

## 対応基準
| 項目 | 基準 |
|------|------|
| 初回返信 | 12時間以内 |
| 解決目標 | 24時間以内 |
| 返品受付 | 30日間（eBayポリシー準拠） |
| 部分返金 | $10以下は即承認 |
| 全額返金 | CEO承認必要（$50超） |

## テンプレート例
- 発送完了通知
- 追跡番号案内
- 到着確認 & レビュー依頼
- 返品手順案内
- 遅延お詫び

## アカウントヘルス管理
- defect rate: 0% 維持
- Late shipment rate: 0% 目標
- Cases closed without seller resolution: 0% 目標

## エスカレーション
- アカウント制限通知 → CEO即報告
- 高額返品（$50超）→ CEO判断
- 詐欺疑い → 証拠保全してCEO報告

## レポート先
CEO（週次）、物流部（配送問題時）
