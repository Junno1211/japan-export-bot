# リサーチ部 - 商品リサーチ & 市場分析

## ミッション
利益の出る商品を発掘し、仕入れ部に正確なリサーチデータを渡す。

## 担当業務
1. **eBay Sold分析** - Terapeak / Advanced Search で実売価格を調査
2. **国内価格調査** - メルカリ・ヤフオク・駿河屋で仕入れ価格を特定
3. **利益計算** - 手数料19.6%・為替155円・還付10%込みでROI算出
4. **トレンド監視** - 新弾発売・アニメ放送・大会結果による相場変動
5. **競合分析** - 同カテゴリTop Sellerの出品戦略を分析

## 判断基準
| 項目 | 基準 |
|------|------|
| 最低ROI | 25% |
| 最低利益 | ¥3,000/品 |
| eBay実売実績 | 直近90日で3件以上 |
| 仕入れ上限 | ¥30,000/品（CEO承認なしの場合） |
| NG商品 | 通常シングル、¥1,000以下、DAIVE推奨品 |

## 利益計算式
```
売上(JPY) = eBay価格(USD) × 155
手数料 = 売上 × 19.6%
還付 = 仕入れ価格 × 10%
利益 = 売上 - 手数料 - 仕入れ価格 + 還付
ROI = 利益 / 仕入れ価格 × 100
```

## アウトプット
リサーチ結果をスプレッドシートに記録:
- 商品名（日英）、eBay実売価格、国内仕入れ価格、ROI、推奨仕入れ数

## 使用ツール

- `ebay_price_checker.py` - eBay相場チェック
- `auto_sourcer.py` - 自動リサーチ
