# ワンピースカーダス部

## 部署ミッション
ワンピースカードダス 1999〜2001 Holo/Prism を中心に、
海外コレクター需要の高い希少カードをソーシングする。

## 仕入れターゲット
- カードダス 1999〜2001 Holo / Prism
- PSA10 プロモカード
- SR / Alt Art / SEC
- Bandai 初期シリーズ（Grand Battle 等）
- フィルムカード / キラカード

## 避けるべき商品
- 通常シングル（ノーマル・コモン）
- ¥1,000以下
- DAIVE推奨品
- 新弾の流通量が多いカード

## 相場が動くタイミング
- アニメ新シリーズ / 映画公開時
- ワンピースカードゲーム新弾発売時
- 海外イベント（Anime Expo等）

## タイトルルール（eBay英語）
```
One Piece [カード名] [年] [Holo/Prism/PSA10] Carddass Bandai Japanese
```

## タイトル混入禁止ワード
- Pokemon / ポケモン系ワード
- Ohtani / 大谷系ワード

## 注意事項
- コンディション表記は必ず具体的に（Near Mint, PSA 10 等）
- ヴィンテージ（1999〜2001）は状態の写真を多めに掲載
- Holo/Prism の光り方が分かる角度で撮影
- 「Carddass」「Bandai」は海外バイヤーの検索ワードなので必須
