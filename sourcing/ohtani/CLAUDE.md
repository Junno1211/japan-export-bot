# 大谷翔平カード部

## 部署ミッション
大谷翔平カードの仕入れ・出品に特化。eBayでの相場を常に意識し、
高ROIのカードを優先的にソーシングする。

## 仕入れターゲット
- RC（ルーキーカード）全般
- PSA/BGS グレーディング済みカード（PSA10優先）
- Auto（直筆サイン）カード
- SR / Alt Art / パラレル
- Topps / Bowman / Panini 系
- 日本限定版（BBM / Epoch / カルビー）

## 避けるべき商品
- ノーマルシングル（レアリティ低）
- Base カード（Insert/Parallelではないもの）
- リプリントやファクシミリサイン

## 相場が動くタイミング
- MLB シーズン中（特にホームラン量産時）
- オールスター / ポストシーズン
- 記録更新時（50-50達成等）
- 日本代表（WBC等）出場時

## タイトルルール（eBay英語）
```
Shohei Ohtani [カード名] [年] [RC/Auto/PSA10] [Topps/Bowman/Panini/BBM] Japanese
```

## タイトル混入禁止ワード
- Pokemon / ポケモン系ワード
- One Piece / ワンピース系ワード

## 注意事項
- コンディション表記は必ず具体的に（Near Mint, PSA 10 等）
- Auto（サイン）カードは真贋に特に注意
- PSAスラブは必ず自撮り画像を使用
- 日本限定カードは「Japan Exclusive」を強調
